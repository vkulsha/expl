Алгоритм создания и внесения полигонов объектов для карты
1. в Планета.SAS открываем карту Google.Satelite, доп.слой Rosreestr для отображения границ земельных участков
2. создаем нужный полигон и сохраняем его в папку соотв объекта (уже созданную ранее)
3. открываем сохраненный файл *.hlg с координатами полигона
4. копируем строки с координатами (без заголовка)
5. открываем Navicat и таблицу ttt
6. вставляем скопированные данные в таблицу ttt в поле f1
7. запускаем в браузере в консоли javascript-функцию createPolygonFromSas(id) с указанием id объекта, для которого был создан полигон
