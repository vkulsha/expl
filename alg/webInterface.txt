Мобильный интерфейс для просмотра, ввода и связи объектов
	Сущности
		Локальная БД
		Интернет БД (копия локальной БД в интернет )
		Локальный серверный процесс синхронизации локальной БД и интернет БД
	
	Интернет БД
		триггер на изменение данных
			установить значение объекта "последнее изменение БД" = объекту "текущее время"

	Локальная БД
		триггер на изменение данных
			установить значение объекта "последнее изменение БД" = объекту "текущее время"
				
	
	Локальный серверный процесс синхронизации локальной БД и интернет БД
		триггер на таймер = 1 раз в мин
			получить из интернет БД объект "последняя синхронизация БД"
			получить из локальной БД объект "последнее изменение БД"
			если объект интернет БД "последняя синхронизация БД" не равен объекту локальной БД "последнее изменение БД"
				выполнить экспорт данных из локальной БД в интернет БД
				установить значение объекта интернет БД "последняя синхронизация БД" = объекту локальной БД "последнее изменение БД"

			получить из локальной БД объект "последняя синхронизация БД"
			получить из интернет БД объект "последнее изменение БД"
			если объект локальной БД "последняя синхронизация БД" не равен объекту интернет БД "последнее изменение БД"
				выполнить импорт данных из интернет БД в локальную БД
				установить значение объекта локальной БД "последняя синхронизация БД" = объекту интернет БД "последнее изменение БД"
